# Amazon Clone 🛒

A simple **Amazon Clone website** built using **HTML and CSS**.  
This project recreates the layout of the Amazon homepage to practice **frontend development and UI design**.

---

## 📌 Project Overview

The Amazon Clone project replicates the basic structure and design of the Amazon homepage.  
It includes a navigation bar, search bar, product sections, and footer similar to the original website.

This project is created for **learning and practice purposes**.

---

## 🚀 Features

- Navigation Bar with Logo
- Search Bar
- Product Sections
- Hero Banner
- Footer Section
- Clean UI Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox

---

## ⚙️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/your-username/amazon-clone.git
```

2. Open the project folder

```
cd amazon-clone
```

3. Run the project

Open **index.html** in your browser.
